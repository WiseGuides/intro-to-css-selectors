Intro to CSS Selectors

There are three different ways of specifying styling. The least common is placing inline, as this p element does.

Slightly more common, CSS rulesets can be placed in <style> tags. This p element gets its styling from a <style> tag placed below.

By far, the most common way to style elements is to write the rulesets in a separate file and then link the CSS file. You can see the link on line 7 of this file.
